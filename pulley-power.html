Pulley System
(Mechanical Advantage)

What is a Pulley?
A pulley is a simple machine used to reduce the effort needed to lift a load. It works by distributing force through ropes.
Key Idea
👉 More supporting ropes → less effort required
Each rope shares part of the load equally.
Formulas
MA = Load / Effort Mechanical Advantage
E = W / n Ideal pulley — n = supporting ropes
W = m × g Weight of load in Newtons
Understanding
  • More pulleys → more supporting ropes
  • More ropes → less effort needed
  • Work is conserved — you pull more distance, but with less force
👉 Insight: You don't reduce the total work done — you only reduce the force needed. The trade-off is pulling a longer rope.
