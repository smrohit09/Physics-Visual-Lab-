📐 Concept Card
Projectile Motion

A projectile moves in two directions at the same time:

  • Horizontal → constant speed
  • Vertical → affected by gravity
Key Idea

These two motions are independent.

  • Gravity does NOT affect horizontal motion
  • Horizontal motion does NOT affect falling
Path

The motion forms a curved path — a parabola.

Controls
  • Velocity → how far it goes
  • Angle → shape of path
  • Gravity → how fast it falls
Insight — at the same speed:
◆Low angle → longer path
◆High angle → higher arc
◆45° → maximum range (flat ground)
